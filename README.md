# playlistGen

scope: website where users can enter a prompt/theme, then log into their spotify and be given a playlist generated based on it 

components needed: 
    - spotify API to build playlist
    - ML to extract songs based on theme 

tools: 
    - front end: react 
    - back-end: python
    - databases: SQL, AWS
